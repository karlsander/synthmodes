Modes

CSAW

Description

CS80-like sawtooth with 'notch'.

Controls


TIMBRE 		notch width

COLOR 		notch depth and polarity


/\/|-_-_

Description

Classic variable wave analog-style oscillator.

Controls


TIMBRE 		triangle to sawtooth to square to pulse

COLOR 		tone, from a 1-pole filter to a waveshaper


/|/|-_-_

Description

Morph from dephased saw to square with pulse width control

Controls


TIMBRE 		dephasing of saw, pulse width of square

COLOR 		waveshape, saw to square


FOLD

Description

Blend of sine/triangle oscillators sent to wavefolder

Controls


TIMBRE 		wavefolder strength

COLOR 		sine -> triangle


_|_|_|_|_

Description

Two detuned variable sine->comb waveforms.

Controls


TIMBRE 		sine -> dirac comb

COLOR 		detune


SYN-_-_

Description

Two square-wave oscillators, one hardsynced to another.

Controls


TIMBRE 		interval between primary/secondary osc

COLOR 		oscillator mix


SYN/|

Description

Two saw oscillators, one hardsynced to another.

Controls


TIMBRE 		interval between primary/secondary osc

COLOR 		oscillator mix

Description

Wavetable synthesis, 1-dimensional scanning through all wavetables.

Controls


TIMBRE 		sweeps wavetables

COLOR 		interpolation method. Past 12oclock, decreases playback resolution.


SAM1

Description

Classic voice synthesizer, configured as a 16-word granular sample player. Trigger plays back word from TIMBRE position to word end.

Controls


TIMBRE 		which 'slice' of the word to play, from start of word to end of word. Apply a ramp shape to play word from start to end.

COLOR 		which word to play


SAM2

Description

Classic voice synthesizer, configured as a 16-word granular sample player. Trigger plays back word from TIMBRE position to word end.

Controls


TIMBRE 		which 'slice' of the word to play, from start of word to end of word. Apply a ramp shape to play word from start to end.

COLOR 		which word to play


NOIS

Description

White noise through a state-variable filter. Pitch input controls filter frequency.

Controls


TIMBRE 		filter resonance

COLOR 		crossfade between lowpass and highpass


TWNQ

Description

White noise processed by two bandpass filters

Controls


TIMBRE 		filter resonance

COLOR 		filter frequency spacing


CLKN

Description

Random samples generated at rate determined by main pitch.

Controls


TIMBRE 		periodicity

COLOR 		quantization, from 2 to 32 distinct values per sample


CLOU

Description

Texture generator by mixing short grains of windowed sine waves

Controls


TIMBRE 		density and overlap of grains

COLOR 		frequency randomization


PRTC

Description

Texture generator by mixing short decaying "pings" of sound

Controls


TIMBRE 		density and overlap of grains

COLOR 		frequency randomization
